# User Guide

## Getting Started

### Create Account
1. Go to https://nexxus-pro.com/register
2. Enter name, email, password
3. Verify email address
4. Complete profile

### Login
1. Enter email and password
2. Check "Remember Me" for persistent login
3. Click Sign In

## Dashboard

### Overview
- **Storage Usage**: Shows used/total storage
- **Recent Files**: Last 5 uploaded files
- **Quick Actions**: Upload, Merge, Share, Settings
- **Activity Log**: Recent actions

### Navigation
- **Dashboard**: Main view
- **Upload**: Upload files
- **Merge**: Merge files
- **Files**: Browse all files
- **Shared**: Files shared with you
- **Settings**: Account settings

## File Management

### Upload Files
1. Click Upload in sidebar
2. Drag & drop or click to browse
3. Select files
4. Wait for upload to complete

### Upload Large Files
- Files >10MB use chunked upload
- Pause and resume supported
- Progress saved if browser closes

### Create Folders
1. Click "New Folder"
2. Enter folder name
3. Folder appears in file list

### Move Files
1. Select file checkbox
2. Click "Move"
3. Choose destination folder
4. Click "Move Here"

### Rename Files
1. Right-click file
2. Select "Rename"
3. Enter new name
4. Press Enter

### Delete Files
1. Select file
2. Click "Delete"
3. Confirm deletion
4. Files go to trash (30 days)

### Star Files
- Click star icon to favorite
- Starred files appear in Starred view
- Click again to remove star

### Search Files
1. Type in search bar
2. Results update in real-time
3. Use filters for advanced search

## File Merging

### Merge PDFs
1. Go to Merge page
2. Add PDF files (min 2)
3. Select output format (PDF)
4. Configure options
5. Click Merge Files

### Merge Images
1. Add images
2. Choose output (PDF or Image)
3. Select layout (Horizontal/Vertical/Grid)
4. Click Merge Files

### Merge Queue
1. Click "Add to Queue"
2. Jobs process in background
3. Check status in Queue tab
4. Download from History

### Merge Options
| Option | Values | Description |
|--------|--------|-------------|
| Page Size | A4, Letter, Legal | Page dimensions |
| Orientation | Portrait, Landscape | Page orientation |
| Compression | Low, Medium, High | File size reduction |
| Quality | 1-100 | Image quality |
| Layout | Horizontal, Vertical, Grid | Image arrangement |

## Sharing

### Share with Users
1. Select file/folder
2. Click Share
3. Enter email addresses
4. Set permissions
5. Set expiration (optional)
6. Send invitation

### Public Links
1. Select file
2. Click "Create Public Link"
3. Copy URL
4. Share with anyone

### Permissions
- **View**: Can only view
- **Edit**: Can edit/rename
- **Download**: Can download
- **Delete**: Can delete (admin only)

## Subscriptions

### Plans

| Plan | Price | Storage | Merges | Features |
|------|-------|---------|--------|----------|
| Free | $0 | 1 GB | 5/month | Basic |
| Basic | $9.99/mo | 10 GB | 50/month | Priority support |
| Pro | $29.99/mo | 100 GB | Unlimited | API, 24/7 support |
| Enterprise | $99.99/mo | 1 TB | Unlimited | Custom solutions |

### Upgrade
1. Go to Pricing
2. Select plan
3. Choose payment method
4. Complete payment
5. Plan activates immediately

### Cancel
1. Settings → Subscription
2. Click Cancel
3. Select reason
4. Confirm
5. Access until period ends

## Payments

### M-Pesa (Kenya)
1. Select M-Pesa
2. Enter phone number (254XXXXXXXXX)
3. Check phone for STK Push
4. Enter PIN
5. Payment completes

### PayPal
1. Select PayPal
2. Click Pay with PayPal
3. Login to PayPal
4. Review and pay
5. Redirected on success

### Bank Transfer
1. Select Bank Transfer
2. Get bank details and reference
3. Transfer amount
4. Upload receipt
5. Wait for verification (24-48 hours)

## Settings

### Profile
- Update name, email, phone
- Change profile picture
- Add company, website, bio

### Notifications
- Email notifications
- Push notifications
- Desktop notifications
- Per-event toggles

### Security
- Change password
- Enable 2FA
- View active sessions
- Revoke suspicious sessions

### API Keys
1. Generate API key
2. Copy key (shown once)
3. Use in API requests
4. Revoke if compromised
